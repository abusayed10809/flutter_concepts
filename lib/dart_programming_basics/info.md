Here is the essential list of data structures every software engineer should know:

## Linear Data Structures

* **Arrays** (Static and Dynamic)
* **Linked Lists** (Singly, Doubly, and Circular)
* **Stacks**
* **Queues** (Standard, Deque, and Priority Queue)

---

## Non-Linear Data Structures

* **Trees** * Binary Trees & Binary Search Trees (BST)
* Balanced Trees (AVL, Red-Black Trees)
* Tries (Prefix Trees)
* Heaps (Max-Heap, Min-Heap)


* **Graphs** (Directed, Undirected, Weighted)

---

## Hash-Based Data Structures

* **Hash Tables** (Hash Maps / Hash Sets)

---

## Advanced / Specialized Data Structures

* **Disjoint Set / Union-Find**
* **Segment Trees / Fenwick Trees** (Binary Indexed Trees)
* **Bloom Filters**
